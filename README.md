# Trusted Mobile Detailing Website

A premium, modern mobile detailing website for Trusted Mobile Detailing in Elk River, Minnesota. Built with React, Vite, and TailwindCSS with brand-matched styling.

## 🎨 Brand Colors

- **Primary Background**: Deep Navy Blue (#0B1C2D)
- **Secondary Background**: Dark Blue Gradient (#0B1C2D to #102A44)
- **Accent Color**: Electric Blue (#1DB7E8)
- **Highlight Accent**: Bright Cyan (#29D3FF)
- **Text Primary**: Metallic Silver (#C8CED6)
- **Text Secondary**: Light Gray (#AAB2BD)

## 🚀 Getting Started

### Installation

```bash
npm install
```

### Development Server

```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## 📄 Pages

- **Home** - Hero section, why choose us, services overview, add-ons highlights
- **Exterior Detailing** - Complete exterior service details and benefits
- **Interior Detailing** - Interior cleaning services and benefits
- **Add-Ons** - Premium add-on services with pricing
- **Gallery** - Before/after showcase
- **Reviews** - Customer testimonials
- **Service Area** - Locations served (Elk River, Otsego, Rogers, Zimmerman, Big Lake)
- **Book Now** - Conversion-focused booking form
- **Contact** - Contact information and message form

## ✨ Features

- **Responsive Design** - Mobile-first approach, works on all devices
- **Sticky Navigation** - Transparent navbar that becomes solid on scroll
- **Smooth Animations** - Fade-in effects, glow effects, shine animations
- **Glass Morphism** - Modern glass-style cards with metallic accents
- **Mobile Floating Call Button** - Quick access to call on mobile devices
- **Form Validation** - Client-side validation on booking form
- **SEO Optimized** - Proper meta tags and semantic HTML

## 🛠️ Tech Stack

- **React 18** - UI framework
- **Vite 5** - Build tool and dev server
- **TailwindCSS 3** - Utility-first CSS framework
- **React Router DOM** - Client-side routing
- **Lucide React** - Modern icon library

## 🔧 Customization

To customize colors, edit `tailwind.config.js`:

```js
colors: {
  'navy-deep': '#0B1C2D',
  'navy-dark': '#102A44',
  'electric-blue': '#1DB7E8',
  'bright-cyan': '#29D3FF',
  'metallic-silver': '#C8CED6',
  'light-gray': '#AAB2BD',
}
```
